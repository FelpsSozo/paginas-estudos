Imagens

Para adicionar uma imagem, use a tag <img>!

Favicon

Para mudar o favicon, use a tag <link>, e ele deve ser feito logo acima do title.

Negrito e Itláico

Para escrever algo em NEGRITO, usamos a tag <strong>!

Para escrever algo em itálico, usamos a tag <em>!

Citações

Para fazermos citações, há duas formas diferentes. Usando a tag <q>, que significa quote - em português citação E a tag <blockquote<, para citações mais longas.

Exemplo de citação curta:

O conhecimento é a chave para o sucesso.

Exemplo de citação longa:

Como disse Nelson Mandela:

"A educação é a arma mais poderosa que você pode usar para mudar o mundo."
