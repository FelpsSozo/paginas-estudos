Links e Ancoras

Para criar um link, use a tag <a>

Exemplo de link para o Google: Google

Exemplo de link para o YouTube: YouTube

Alguns parametros são bem importantes para links, como o target, e o rel. o target uasdo para se referir a link externo ou interno. E o rel para referenciar qual página é. Links externos precisam da URL completa, e os intenos só precisam da referência do local do arquivo.

Tenha em mente que, ao definir links nos códigos, é preciso saber como procurar, por isso, é bom estudar linux.
